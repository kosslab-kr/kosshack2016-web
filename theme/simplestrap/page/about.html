About

심플스트랩 테마를 사용하고 있는 블리티터 입니다.

블리티터의 팀블로그 모듈을 중심으로 꾸며진 예제로 회원관리, 블로그 포스트 관리, 방명록 관리 등의 기능을 제공하고 있습니다.


Blititor

OLD STYLE WEB SITE TEMPLATE for DEVELOPERS and WEB AGENCIES


블리티터는 다른 솔루션에 비해 다음과 같은 장점을 가지고 있습니다.

  • Template 적절하게 즉시 사용할 수 있는 다양한 템플릿 을 제공합니다.
  • Structure 구조 단순해 수정하기 쉽습니다.
  • Familiar 기존 경험을 그대로 적용해 사용할 수 있습니다.
  • All in One 왠만한 모듈은 다 제공합니다.(그냥 쓰실 수 있는 퀄리티)

여기에 추가로 개발자들이 고려할 내용을 소개합니다.

  • Node.JS 더 이상 PHP 를 사용하지 않아도 됩니다.(자바스크립트 만세)
  • Callback 이해하기 쉽게 콜백 패턴을 사용합니다.(Pass Promise, Wait Async)
  • MySQL 이미 배운 쿼리문을 그대로 사용합니다.(물론 MariaDB 호환)
  • Serverside Template 클라이언트 사이드 기술 과부하 솔루션이 아닙니다.
  • AJAX 부분적으로 웹소켓과 AJAX 를 사용합니다.(jQuery == 사랑)
